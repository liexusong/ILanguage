The following LOGICAL data types are available to the programmer.
The relevant file is listed next to each one that is documented.

Constants: applying as a function gives the constant back; attempting
    to map gives a domain error.
  Errors
  Numbers (arithmetic.txt)
  Characters

Functions: mapping gives a fork (see forks.txt).
  Builtins
  Undefined names
  Quasiquotes

Collections: mapping applies to each element in the list and returns the
    appropriate collection of results.
  Lists (lists.txt)
